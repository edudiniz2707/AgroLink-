AgroLink v6 — Portal Brasil (pronto para GitHub Pages)

Conteúdo:
- index.html (portal completo)
- style.css (estilos)
- app.js (lógica do cliente usando localStorage)
- images/ (imagens otimizadas para web)

Publicação no GitHub Pages:
1. Crie um repositório no GitHub (ex: agrolink-portal).
2. Faça upload de todo o conteúdo deste diretório (descompacte o ZIP e envie).
3. Vá em Settings → Pages → Deploy from branch → escolha 'main' e root (/).
4. Acesse: https://SEU-USUARIO.github.io/NOME-DO-REPO/

Observações:
- O site funciona offline usando localStorage. Para ter login real e armazenamento centralizado, conecte a um back-end (Firebase, Supabase, ou API personalizada).
- Se quiser, eu faço o passo-a-passo com comandos git ou preparo deploy direto no Vercel.
